# Support

## What This Repo Covers

- staged repo boundary issues
- doc and proof navigation issues
- package import/layout issues inside the staged repo

## What Is Deferred

- Phase 4.5 performance augmentation

## How To Report Problems

Report with:

- exact file or path
- exact command if a command was run
- stdout or stderr if relevant
- whether the problem is repo-boundary, proof-integrity, or operator-only dependency related
