# Legal Boundaries

This repo is a private staging surface for ZPE Geo. It is not a public-release representation of final launch state.

## Boundary Rules

- repo-local code and proof material are inside this repo
- operator-only data, secrets, and vendored runtime payloads stay outside the repo
- preserved proof artifacts outrank summary prose when they disagree
- unresolved contradictions remain explicit

## Dataset And Resource Notes

- `code/fixtures/` contains only lightweight deterministic fixtures
- larger external samples remain outside the repo under the outer workspace
- raw `third_party/` remains outside the repo until dependency policy is finalized
- archived proof artifacts may refer to resources or absolute paths that are not portable

## Current Integrity Limits

The staged repo preserves the known contradiction set:

- broken `/Users/prinivenpillay/...` paths in historical bundle content
- max-wave pass signals that conflict with the max-claim map and residual risk register
- RunPod requirement disagreement between the gap matrix and readiness manifest

Those are evidence problems, not documentation problems.
